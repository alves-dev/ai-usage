# Compatibility Matrix

The table below describes the compatibility between the integration versions and supported Home Assistant core releases.

| Integration Version | HA Version  Supported    | HA Version Not Supported |
|---------------------|--------------------------|--------------------------|
| 0.0.1               | 2026.4 - 2026.5 - 2026.6 | ---                      |

**Note:** Versions not listed in the table indicate that they have not been tested, so they may or may not work.

Run the version-specific validation script before updating this matrix:

```bash
scripts/validate-homeassistant-version.sh 2026.x.x
```
